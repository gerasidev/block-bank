## ğŸ› The Banking Mechanism (Fractional Reserve)

$THYSEAS utilizes a **10x Leverage Engine** in the `LendingVault`:
1. **Lenders Deposit Liquidity**: Lenders provide the base reserve (ETH/BTC) into the vault.
2. **Borrowers Lock Collateral**: Physical assets (Robots/Real Estate) are appraised and locked.
3. **Militated Fractional Minting**: The bank can mint up to 10x the value of the locked collateral, provided the circulating supply does not exceed 10x the vault's total liquid reserves.
4. **Manual Release**: All liquidity releases require a 2-step manual audit and approval flow.
